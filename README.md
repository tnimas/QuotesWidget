# Quotes Widget

A customizable quote widget for [Lively Wallpaper](https://www.rocksdanister.com/lively/) that displays rotating quotes on your desktop.

## Features

- Display quotes from a custom list
- Configurable update intervals: daily, hourly, or every minute
- Adjustable font size and text color
- Glassmorphism design with blur effect

## Installation

1. Download or clone this repository
2. In Lively Wallpaper, click the **+** button to add a new wallpaper
3. Select the `index.html` file from this folder

## Configuration

Right-click the wallpaper in Lively and select **Customize** to access settings:

| Setting | Description |
|---------|-------------|
| Quotes | Enter quotes in format: `Quote text \| Author` (one per line) |
| Update Interval | How often the quote changes (Daily/Hourly/Every Minute) |
| Font Size | Adjust text size (14-100px) |
| Text Color | Change the quote text color |

## Example Quotes Format

```
Stay hungry, stay foolish. | Steve Jobs
The only way to do great work is to love what you do. | Steve Jobs
Innovation distinguishes between a leader and a follower. | Steve Jobs
```
